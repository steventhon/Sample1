Bootstrap

Bootstrap has its own CSS and starter components, such as this Jumbotron template, that helped me develop the webpage design both easier and faster.

Font Awesome

Font Awesome has many icons, such as the flag and GitHub ones I've used below, that are free to use. This saves me the time and effort of trying to make my own. Now I can focus, more importantly, on the webpage design and content and still have professional-quality icons.

GitHub

GitHub lets me show my work to the public, saves my work online, and allows for collaboration on my projects. This webpage is stored on my GitHub and I can work on it from any computer with internet access.
